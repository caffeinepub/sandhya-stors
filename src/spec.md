# Specification

## Summary
**Goal:** Add a URL shortener feature to the Sandhya Stors homepage that allows users to convert long URLs into shortened versions.

**Planned changes:**
- Add a new URL shortener section on the homepage with an input field for pasting long URLs
- Implement backend functionality to generate unique short codes and store URL mappings
- Create redirect handling so short URLs redirect to their original destinations
- Display generated shortened URLs with a copy-to-clipboard button for easy sharing
- Maintain the existing warm, earthy design aesthetic throughout the new section

**User-visible outcome:** Users can paste a long URL on the Sandhya Stors homepage, receive a shortened version, and copy it to share. The shortened URLs will redirect to the original destinations when accessed.
